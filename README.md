# Case Study 01 – El Ni

Dieses Repository enthält die Aufgabenlösung für die Case Study 01 im Rahmen des Softwaredesign-Kurses.

## Installation & Setup

### 1) Repository klonen
git clone <REPO-URL>

### 2) Virtuelle Umgebung erstellen
python3 -m venv .venv

### 3) Virtuelle Umgebung aktivieren (macOS/Linux)
source .venv/bin/activate

### 4) Abhängigkeiten installieren
pip install -r requirements.txt

### 5) Projekt ausführen
python main.py

## Projektstruktur
- main.py – Startpunkt des Projekts („Hello World“)
- requirements.txt – externe Python-Abhängigkeiten
- README.md – Projektdokumentation
